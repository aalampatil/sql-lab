# 📘 Labs Assignment

A full-stack labs assignment repository containing both frontend and backend code.  
This project is structured to separate server-side logic (backend) from client-side code (frontend), making development and deployment easier.

## 🗂 Repository Structure

- **backend/** – Contains server side code (written in Node.js/Express).
- **frontend/** – Contains the frontend web application (vite React).

## Getting Started

These instructions will help you set up and run the project on your local machine for development and testing.

### Prerequisites

Make sure you have **Node.js** and **npm** installed:

```bash
node --version
npm --version

git clone url

cd backend
npm install
npm run start

cd frontend
npm install
npm run dev
```
